# 设计说明：ui-all-recipes-card-fixed-thumb-v1

## 范围

仅优化全部菜谱页（`AllRecipesView`）卡片版式，不改其他页面卡片。

## 布局规则

- 卡片左侧缩略图固定尺寸：`96x96`，使用 `object-fit: cover`。
- 卡片右侧文案区：
  - 标题单行省略
  - 简介双行省略（存在时显示）
- 卡片最小高度固定，保持列表视觉整齐。

## 样式策略

- 使用 `.recipe-directory` 作用域样式，避免影响首页和其他列表卡片。
- 在 `cards.css` 内新增限定选择器（仅作用于全部菜谱列表）。

## 非目标

- 不引入瀑布流/小红书式双列卡片。
- 不调整筛选、排序、搜索逻辑。
