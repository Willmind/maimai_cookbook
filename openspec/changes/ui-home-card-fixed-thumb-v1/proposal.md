# 变更提案：ui-home-card-fixed-thumb-v1

## 为什么

首页包含“搜索结果 / 做饭记录流 / 等着开火”等多个卡片区域。当前这些区域使用通用的 `.recipe-card` 样式，缩略图与文字内容可能导致卡片高度不一致、视觉节奏不够整齐。

## 变更内容

- 在首页对“可复用卡片列表容器”增加容器类 `home-fixed-thumb`
- 对该容器下的卡片：
  - 缩略图固定为 `96x96` 的方图
  - 标题单行省略
  - 内容/简介双行省略（基于 `.muted` 文本）

## 影响范围

- 仅影响首页展示层样式（不改变筛选/路由/数据）。
