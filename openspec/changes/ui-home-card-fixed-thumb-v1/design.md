# 设计说明：ui-home-card-fixed-thumb-v1

## 适用范围

仅针对首页：

- 搜索结果分组（`搜索结果 -> 菜谱 / 做饭记录`）
- `做饭记录流`
- `等着开火`

## 设计规则

- 使用容器类 `home-fixed-thumb` 作为作用域
- 在作用域内对 `.recipe-card` 做覆盖：
  - 左侧缩略图固定 `96x96`，并裁切 `object-fit: cover`
  - 标题 `h3` 限制为单行省略
  - 卡片内容 `.muted` 限制为最多两行省略

## 非目标

- 不改动其他页面的卡片样式
- 不改变数据结构和交互逻辑
